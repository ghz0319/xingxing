## 🧩 How It Works

这个仓库的核心功能由 GitHub Actions 实现，自动执行以下步骤 👇：

1. 使用 `requests` 获取你的 Star 列表；
2. 根据语言自动分组；
3. 更新并生成 `stars.md` 文件；
4. 自动提交回仓库。

📦 一旦配置好 `GH_TOKEN`，整个过程将全自动运行，每日更新。
